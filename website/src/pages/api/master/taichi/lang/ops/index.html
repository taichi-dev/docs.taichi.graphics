Copy and cast a scalar or a matrix to a specified data type. Must be called in Taichi scope.

Parameters

  • obj (Union[primitive_types, Matrix]) â€“ Input scalar or matrix.

  • dtype (types) â€“ A primitive type defined in primitive_types.

Returns

Example:

>>> @ti.kernel
>>> def test():
>>>     x = ti.Matrix([0, 1, 2], ti.i32)
